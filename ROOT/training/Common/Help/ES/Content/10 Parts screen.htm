Pantalla Recambios

La pantalla Recambios se utiliza para localizar recambios, agregar recambios a la Lista de pedidos y transferir recambios a su Sistema de administración de concesionario (DMS).

Cuando identifique un vehículo, se mostrará automáticamente la pantalla Recambios. Para acceder a la pantalla de Recambios, pulse Recambios en la parte superior de la pantalla.

 

Desde la página de Recambios puede:

  • Buscar recambios utilizando una gama de tipos de búsqueda
  • Navegar por el índice usando una vista gráfica o de texto
  • Usar la herramienta de navegación rápida
  • Mostrar las imágenes de diferentes maneras
  • Trabajar con la Cuadrícula de datos de recambios
  • Agregar notas de usuario a una sección o un recambio
  • Agregar recambios a la Lista de pedidos
  • Transferir un recambio a su DMS

 

Presione F11 en el teclado para mostrar la vista de pantalla completa de Microcat.
