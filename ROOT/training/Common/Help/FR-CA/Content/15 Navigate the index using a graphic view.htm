Parcourir l'Index en utilisant les illustrations

L'Index graphique est un index imagé des sections d'un véhicule.

 

Pour parcourir l'Index en utilisant les illustrations :

  1. Cliquez sur l'icône pour afficher une vue graphique.
  2. Cliquez sur une section principale de l'Index.
  3. Cliquez sur une section secondaire de l'Index.
  4. Parcourez l'Index jusqu'à l'image désirée.
  5. À partir de l'image, vous pouvez :
  • Double-cliquer sur la référence d'une pièce pour l'ajouter à la Liste de commandes.
  • Cliquer sur la référence d'une pièce pour afficher la fenêtre Informations pièce.
  • Cliquer sur l'onglet Pièces illustrées pour afficher une liste des pièces présentes dans l'image.

 

Pour afficher la fenêtre Pièces illustrées, cliquez sur l'icône dans la barre de titre.
