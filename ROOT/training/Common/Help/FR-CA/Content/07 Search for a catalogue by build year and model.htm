Rechercher un catalogue selon l'année de construction et le modèle

La Recherche de catalogue permet de rechercher un véhicule selon l'année de construction et le modèle. Vous pouvez ensuite affiner votre recherche en sélectionnant les principaux attributs du véhicule.

La recherche de catalogue est une fonctionnalité utile si vous ne possédez pas le VIN d'un véhicule. Plus vous indiquez de caractéristiques majeures, plus votre recherche de pièces sera précise.

 

Pour rechercher un catalogue :

  1. Sélectionnez l'année de construction dans la liste déroulante Année.
  2. Sélectionnez le catalogue dans la liste déroulante Catalogue.
  3. Sélectionnez les caractéristiques majeures dans les listes déroulantes situées dans la fenêtre Renseignements sur le véhicule.

Utilisez la fenêtre Recherche de pièces ou l'Index graphique pour rechercher des pièces.

 

Pour rétablir les caractéristiques par défaut, cliquez sur Réinitialiser.
