Rechercher des pièces à partir du code de pièce du constructeur

Il est possible de rechercher des pièces de différentes façons, y compris par numéro de pièce ou par code de pièce du constructeur.

Les types de recherche varient selon le constructeur. Par exemple, vous pouvez effectuer une recherche par numéro de base, code FINIS, PNC ou code d'article.

 

Pour rechercher des pièces à partir du code de pièce du constructeur :

  1. Saisissez le code de pièce du constructeur dans le champ Rechercher par.

Par exemple, vous pouvez saisir un numéro de base, un code FINIS, un PNC ou un code d'article.

  1. Sélectionnez le type de recherche correspondant.
  2. Cliquez sur l'icône ou appuyez sur la touche Entrée du clavier.

Les résultats de la recherche s'affichent.

  1. À partir des résultats de la recherche, vous pouvez :
  • Sélectionnez la case pour ajouter la pièce à la Liste de commandes.
  • Cliquer sur l'icône pour visualiser la pièce dans l'image.
  • Cliquer sur la description de la pièce pour afficher la fenêtre Informations pièce.

 

Pour revenir aux résultats de la recherche à partir d'une image, cliquez sur l'onglet Résultats de la recherche.
