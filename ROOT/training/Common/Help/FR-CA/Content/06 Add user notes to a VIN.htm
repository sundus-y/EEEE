Ajouter des notes d'utilisateur à un VIN

Microcat permet d'ajouter des notes d'utilisateur à un VIN.

Les notes de VIN aident à enregistrer l'historique d'un véhicule particulier. Vous pouvez par exemple ajouter une note lorsque des jantes en alliage ont été montées sur le véhicule d'un client.

 

Pour ajouter une note de VIN :

  1. Identifiez un véhicule par VIN.
  2. Cliquez sur Ajouter note au VIN dans la fenêtre Renseignements sur le véhicule.
  3. Saisissez le titre et le texte de la note.
  4. Cliquez sur Soumettre.

L'icône s'affiche près du VIN. Déplacez la souris sur l'icône pour afficher la note.

 

Pour modifier une note de VIN :

  1. Cliquez sur l'icône affichée près du VIN.
  2. Cliquez sur Modifier.
  3. Modifiez le texte.
  4. Cliquez sur Soumettre.

 

Pour supprimer une note de VIN :

  1. Cliquez sur l'icône affichée près du VIN.
  2. Cliquez sur Modifier.
  3. Cliquez sur Supprimer.

 

Pour ajouter une note à un catalogue, cliquez sur Ajouter note catalogue dans la fenêtre Renseignements sur le véhicule.
