Rechercher des pièces par description

Il est possible de rechercher des pièces de différentes façons, y compris par description. Le filtre permet de filtrer les résultats de la recherche au fur et à mesure que vous tapez.

 

Pour rechercher des pièces par description :

  1. Saisissez une description dans le champ Rechercher par.

La description doit être exactement la même que celle utilisée par le constructeur dans le catalogue.

  1. Assurez-vous que le type de recherche sélectionné est Description.
  2. Cliquez sur l'icône ou appuyez sur la touche Entrée du clavier.

Les résultats de la recherche s'affichent.

  1. Pour réduire le nombre de résultats, saisissez un critère de filtre dans le champ de filtrage.

Le critère peut être composé d'un ou de plusieurs mots. Seuls les résultats contenant le critère de filtre s'affichent.

  1. À partir des résultats de la recherche, vous pouvez :
  • Sélectionnez la case pour ajouter la pièce à la Liste de commandes.
  • Cliquer sur l'icône pour visualiser la pièce dans l'image.
  • Cliquer sur la description de la pièce pour afficher la fenêtre Informations pièce.

 

Pour afficher ou masquer les renseignements sur le véhicule, cliquez sur l'icône .
