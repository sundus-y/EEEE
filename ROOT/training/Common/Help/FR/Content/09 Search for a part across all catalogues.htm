Rechercher une pièce dans tous les catalogues

Microcat permet de rechercher une pièce dans tous les catalogues.

La recherche croisée dans plusieurs catalogues est utile pour vérifier sur quels véhicules une pièce peut être montée. Vous pouvez également vérifier la description et le prix (si disponible) de la pièce.

 

Pour rechercher une pièce dans tous les catalogues :

  1. Tapez le numéro de la pièce dans le champ Rechercher par.
  2. Assurez-vous que le type de recherche sélectionné est Numéro de pièce.
  3. Assurez-vous que la case Catalogue croisé est cochée.
  4. Cliquez sur l'icône ou appuyez sur la touche Entrée du clavier.

Les véhicules sur lesquels la pièce peut être montée s'affichent.

  1. Sélectionnez un véhicule pour accéder à la pièce.
