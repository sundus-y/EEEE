Modifier ou masquer les prix

L'écran de traitement des commandes permet de modifier le niveau de prix appliqué aux pièces dans une Liste de commandes. Vous pouvez également masquer le prix des pièces sur la Liste de commandes. Il est utile de masquer le prix des pièces lorsque vous confirmez une pièce directement auprès du client.

La disponibilité des informations sur les prix et les niveaux de prix varie selon le constructeur.

 

Pour modifier le niveau de prix :

  1. Affichez une Liste de commandes.
  2. Sélectionnez un niveau de prix dans la liste déroulante Prix unitaire.

Le niveau de prix sélectionné est appliqué à toutes les pièces dans la Liste de commandes.

 

Pour masquer le prix des pièces :

  1. Affichez une Liste de commandes.
  2. Sélectionnez Masquer les prix dans la liste déroulante Prix unitaire.

Les colonnes de la Liste de commandes contenant les prix sont masquées. Les prix sont également masqués dans la grille de données des pièces ainsi que dans la fenêtre Informations pièce.

  1. Pour afficher les prix, sélectionnez un niveau de prix dans la liste déroulante Prix unitaire.

 

Pour voir les options d'affichage des prix, cliquez sur Paramètres généraux dans le menu Paramètres.
