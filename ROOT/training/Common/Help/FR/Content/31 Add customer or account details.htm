Ajouter les renseignements sur le client ou sur son compte

Les informations sur le client ou sur son compte peuvent être ajoutées à la Liste de commandes.

Les informations sur le client sont affichées sur les sorties imprimante, sur les courriers électroniques et sur les commandes sauvegardées. Cela permet de donner à vos documents un aspect plus professionnel.

 

Pour ajouter les informations sur le client :

  1. Saisissez le nom du client dans le champ Nom du client.
  2. Saisissez le numéro du compte ou la référence dans les champs correspondants.
  3. Cliquez sur l'onglet Client pour ajouter les coordonnées du client.

 

Pour sauvegarder la Liste de commandes, ajoutez le nom du client puis cliquez sur Sauvegarder. Pour récupérer une Liste de commandes sauvegardée, cliquez sur Commandes sauvegardées puis sélectionnez la Liste de commandes voulue.
