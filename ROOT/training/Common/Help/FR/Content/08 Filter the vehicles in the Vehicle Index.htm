Filtrer les véhicules dans l'Index des véhicules

L'Index des véhicules présente tous les véhicules selon un affichage graphique ou textuel. Le filtre permet de filtrer les noms des véhicules au fur et à mesure que vous tapez.

 

Pour filtrer les véhicules :

  1. Saisissez le critère de filtre, tel que le nom et/ou l'année du véhicule dans le champ de filtrage.

Le critère peut être composé d'un ou de plusieurs mots. Seuls les véhicules contenant le critère s'affichent.

  1. Sélectionnez un véhicule à partir de l'index ou dans la liste déroulante du filtre.
  2. Sélectionnez les caractéristiques majeures dans les listes déroulantes situées dans le panneau Renseignements sur le véhicule.

Plus vous indiquez de caractéristiques majeures, plus votre recherche de pièces sera précise.
