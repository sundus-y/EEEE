Rechercher un véhicule par VIN ou VIN partiel

La recherche par VIN est la méthode d'identification d'un véhicule la plus fiable qui permet de sélectionner les pièces de façon précise.

Les types de recherche disponibles varient selon le constructeur. Vous pouvez, par exemple, effectuer une recherche par numéro d'immatriculation, numéro de modèle et numéro de châssis.

 

Pour effectuer une recherche par VIN :

  1. Saisissez le VIN dans le champ Rechercher par.

Vous pouvez également sélectionner un VIN dans la liste déroulante contenant les dernières recherches de VIN.

  1. Cliquez sur Rechercher ou appuyez sur la touche Entrée du clavier.

Le catalogue est chargé et les informations sur le véhicule s'affichent.

Utilisez le panneau Recherche de pièces ou l'Index graphique pour rechercher des pièces.

 

Pour effectuer une recherche par VIN partiel :

  1. Saisissez les 8 derniers caractères du VIN dans le champ Rechercher par.

Le compteur de caractères situé près du champ d'entrée du VIN indique le nombre de caractères ayant été saisis.

  1. Cliquez sur Rechercher ou appuyez sur la touche Entrée du clavier.
  2. Si plusieurs VIN s'affichent, sélectionnez le VIN requis.

Le catalogue est chargé et les informations sur le véhicule s'affichent.

Utilisez le panneau Recherche de pièces ou l'Index graphique pour rechercher des pièces.

 

Pour imprimer ou envoyer les renseignements sur le véhicule que vous avez identifié cliquez sur Imprimer / Envoyer courriel.

Pour afficher le panneau Recherche de véhicule, cliquez sur l'icône dans la barre de titre.
