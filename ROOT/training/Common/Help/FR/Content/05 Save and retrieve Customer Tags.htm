Sauvegarder et récupérer des Identifications client

Les Identifications client permettent de sauvegarder un véhicule spécifique sous un nom unique et sont particulièrement utiles pour les clients réguliers, tel que les clients possédant une flotte de véhicules.

 

Pour sauvegarder une Identification client :

  1. Identifiez un véhicule par VIN.
  2. Indiquez l'Identification client dans le champ Identification client, situé dans le panneau Renseignements sur le véhicule.
  3. Cliquez sur Sauvegarder.

 

Pour récupérer une Identification client :

  1. Indiquez l'Identification client dans le champ Identification client, situé dans le panneau Recherche de véhicule.
  2. Cliquez sur Rechercher ou appuyez sur la touche Entrée du clavier.

Le catalogue est chargé et les informations sur le véhicule s'affichent.

 

Pour accéder à une liste complète de vos Identifications client, cliquez sur l'icône , dans le panneau Recherche de véhicule.
