Imprimer ou envoyer une Liste de commandes par courrier électronique

Avec Microcat, vous pouvez imprimer ou envoyer une Liste de commandes par courrier électronique.

Envoyez la Liste de commandes par courrier électronique afin de confirmer une commande avec un client ou d'optimiser vos opérations quotidiennes.

 

Pour imprimer une Liste de commandes :

  1. Ajoutez les renseignements sur le client ou sur son compte.

Pour ajouter plus de renseignements sur le client, cliquez sur l'onglet Client et ajoutez les renseignements manquants. Ces informations seront reportées sur la sortie imprimante.

  1. Cliquez sur Imprimer.

Un aperçu avant impression s'affiche.

  1. Cliquez sur l'icône .

Si nécessaire, envoyez la sortie imprimante à votre client par fax.

 

Pour envoyer une Liste de commandes par courrier électronique :

  1. Ajoutez les renseignements sur le client ou sur son compte.

Pour ajouter plus de renseignements sur le client, cliquez sur l'onglet Client et ajoutez les renseignements manquants. Ces informations seront reportées dans le courrier envoyé.

  1. Cliquez sur Envoyer courriel.

La fenêtre Envoyer courriel s'affiche.

  1. Saisissez les informations à inclure dans le courrier électronique telles que l'adresse électronique, l'objet et le message.

Pour que la réponse du client soit envoyée à une adresse électronique différente, saisissez cette adresse dans le champ Répondre à. Par exemple, saisissez l'adresse de votre entreprise ou de votre service.

  1. Cliquez sur Envoyer courriel.
  2. Cliquez sur OK.

 

Pour ajouter un en-tête et un pied de page, cliquez sur Paramètres généraux dans le menu Paramètres puis saisissez le texte à afficher dans les champs correspondants.

Pour masquer les numéros de pièce, cliquez sur Paramètres généraux dans le menu Paramètres puis sélectionnez la case Masquer les numéros de pièce.
