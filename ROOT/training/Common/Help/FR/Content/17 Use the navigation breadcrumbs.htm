Utiliser le fil d'Ariane

Le fil d'Ariane, situé en haut de l'écran, est composé de liens qui permettent de naviguer facilement entre les sections d'un véhicule.

Le fil d'Ariane permet de se situer au sein du catalogue et de visualiser le chemin à suivre pour revenir à la page d'accueil.

 

Pour utiliser le fil d'Ariane :

  1. Recherchez une image.
  2. Déplacez la souris sur un des liens du fil d'Ariane.

Une liste déroulante affiche les sections.

  1. Sélectionnez une section dans la liste déroulante.

 

Pour revenir à la page d'accueil à n'importe quel moment, cliquez sur Accueil.
