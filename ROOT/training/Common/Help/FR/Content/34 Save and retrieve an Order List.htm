Sauvegarder et récupérer une Liste de commandes

La Liste de commandes peut être sauvegardée puis récupérée ultérieurement.

Cela permet de servir plusieurs clients simultanément. Vous pouvez également utiliser cette fonctionnalité pour retrouver les devis en attente d'approbation à chaque fin de mois et relancer vos clients.

 

Pour sauvegarder une Liste de commandes :

  1. Ajoutez les renseignements sur le client ou sur son compte.

Pour ajouter plus de renseignements sur le client, cliquez sur l'onglet Client et ajoutez les renseignements manquants.

  1. Cliquez sur Sauvegarder.

La Liste de commandes est sauvegardée dans le panneau Commandes sauvegardées.

 

Pour récupérer une Liste de commandes sauvegardée :

  1. Cliquez sur Commandes sauvegardées au-dessus de la Liste de commandes.
  2. Cliquez sur la Liste de commandes requise.

La Liste de commandes s'affiche dans l'onglet Commande. Les renseignements sur le client s'affichent dans l'onglet Client.

 

Pour supprimer une Liste de commandes sauvegardée, retrouvez la Liste de commandes puis cliquez sur Supprimer en bas de la Liste de commandes.
