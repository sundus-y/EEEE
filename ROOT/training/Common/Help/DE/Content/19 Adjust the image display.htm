Bildanzeige anpassen

Die Bildanzeige können Sie so anpassen, dass sie Ihren Vorgaben entspricht. Darüber hinaus können Sie das Bild drucken oder per E-Mail senden.

 

Verwenden Sie die folgenden Optionen, um die Bildanzeige anzupassen:

Bild in tatsächlicher Größe anzeigen

Bild an Seitenbreite anpassen

Bild an Seitenhöhe anpassen

Bild drucken

Bild per E-Mail senden

Bild an eine beliebige Stelle ziehen. Vergewissern Sie sich, dass Sie auf das Symbol klicken, um die Option zu deaktivieren.

Dieser Baugruppe eine Notiz hinzufügen

 

Um das nächste bzw. vorherige Bild im Katalog aufzurufen, klicken Sie auf das Symbol .
