Teilesuche anhand von Herstellercodes ausführen

Sie haben die Möglichkeit, nach Teilen anhand verschiedener Sucharten zu suchen. Dazu zählen u. a. die Teilenummer und der Herstellercode.

Die Sucharten unterscheiden sich abhängig von Ihrem Hersteller. Beispiel: Grundnummer, FINIS-Nr., PNC oder Warennummer.

 

So suchen Sie nach Teilen anhand von Herstellercodes:

  1. Geben Sie im Feld Suche nach den Herstellercode ein.

Beispiel: Grundnummer, FINIS-Nr., PNC oder Warennummer.

  1. Wählen Sie die entsprechende Suchart aus.
  2. Klicken Sie auf das Symbol oder drücken Sie auf der Tastatur auf Enter.

Die Suchergebnisse werden angezeigt.

  1. Über die Suchergebnisse können Sie Folgendes ausführen:
  • Aktivieren Sie das Kontrollkästchen, um das Teil der Bestellliste hinzuzufügen.
  • Klicken Sie auf das Symbol , um das Teil im Bild anzuzeigen.
  • Klicken Sie auf die Teilebeschreibung, um das Fenster Teiledaten anzuzeigen.

 

Um vom Bild zu den Suchergebnissen zurückzukehren, klicken Sie auf die Registerkarte Suchergebnisse.
