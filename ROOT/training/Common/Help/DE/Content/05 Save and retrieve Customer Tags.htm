Kundenkennzeichen speichern und abrufen

Mithilfe von Kundenkennzeichen können Sie für ein bestimmtes Fahrzeug einen eindeutigen Namen speichern. Außerdem sind sie hilfreich bei regelmäßigen Kunden, beispielsweise Großkunden.

 

So speichern Sie ein Kundenkennzeichen:

  1. Identifizieren Sie ein Fahrzeug anhand der Fg.-Nr.
  2. Geben Sie das Kundenkennzeichen im Feld Kundenkennzeichen ein, das sich im Fenster Fahrzeugangaben befindet.
  3. Klicken Sie auf Speichern.

 

So rufen Sie ein Kundenkennzeichen ab:

  1. Geben Sie das gespeicherte Kundenkennzeichen im Feld Kundenkennzeichen ein, das sich im Fenster Fahrzeugsuche befindet.
  2. Klicken Sie auf Suchen oder drücken Sie auf der Tastatur auf Enter.

Der Katalog wird geladen und die Fahrzeugangaben werden angezeigt.

 

Um eine vollständige Liste Ihrer Kundenkennzeichen abzurufen, klicken Sie im Fenster Fahrzeugsuche auf das Symbol .
