Teilesuche anhand einer Beschreibung ausführen

Sie haben die Möglichkeit, nach Teilen anhand verschiedener Sucharten zu suchen. Dazu zählt u. a. die Teilebeschreibung. Mithilfe des intuitiven Filters können Sie die Suchergebnisse während Ihrer Eingabe filtern.

 

So suchen Sie nach Teilen anhand einer Beschreibung:

  1. Geben Sie im Feld Suche nach eine Beschreibung ein.

Die eingegebene Beschreibung muss mit der Beschreibung übereinstimmen, die vom Hersteller im Katalog verwendet wurde.

  1. Vergewissern Sie sich, dass als Suchart die Option Beschreibung ausgewählt ist.
  2. Klicken Sie auf das Symbol oder drücken Sie auf der Tastatur auf Enter.

Die Suchergebnisse werden angezeigt.

  1. Wenn Sie Ihre Suchergebnisse einschränken möchten, geben Sie Ihre Filterkriterien in das Filterfeld ein.

Die Filterkriterien können aus einem einzelnen Wort oder aus mehreren Wörtern bestehen. Es werden nur die Suchergebnisse angezeigt, auf die die Filterkriterien zutreffen.

  1. Über die Suchergebnisse können Sie Folgendes ausführen:
  • Aktivieren Sie das Kontrollkästchen, um das Teil der Bestellliste hinzuzufügen.
  • Klicken Sie auf das Symbol , um das Teil im Bild anzuzeigen.
  • Klicken Sie auf die Teilebeschreibung, um das Fenster Teiledaten anzuzeigen.

 

Um Fahrzeugangaben anzuzeigen oder auszublenden, klicken Sie auf das Symbol .
