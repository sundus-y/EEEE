Schnelle Navigation verwenden

Die schnelle Navigation befindet sich am oberen Bildschirmrand und ermöglicht Ihnen die Eingabe von Tastaturbefehlen, um zwischen Katalogen oder Baugruppen zu wechseln.

Die schnelle Navigation ist sowohl für neue als auch erfahrene Benutzer nützlich.

 

Tastaturbefehl

Wechseln zu

<.>+<Katalogcode>+<Enter>

Katalog über einen Katalogcode

<.>+<die letzten acht Zeichen der Fg.-Nr.>+<.>+<Enter>

Katalog über eine Fg.-Nr.

Baugruppennummer+<Enter>

Baugruppe innerhalb eines Katalogs

 

So verwenden Sie die schnelle Navigation:

  1. Geben Sie im Feld für die schnelle Navigation einen Tastaturbefehl ein.
  2. Klicken Sie auf das Symbol oder drücken Sie auf der Tastatur auf Enter.

Der Katalog, die Baugruppe oder die Suchergebnisse wird/werden angezeigt.

  1. Wenn mehrere Ergebnisse angezeigt werden, wählen Sie die gewünschte Option aus.

 

Bewegen Sie die Maus über das Symbol , um eine vollständige Liste der Tastaturbefehle anzuzeigen.
