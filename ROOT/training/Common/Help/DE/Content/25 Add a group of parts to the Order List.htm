Der Bestellliste eine Teilegruppe hinzufügen

Mithilfe des Lasso-Werkzeugs können Sie eine Gruppe von verschiedenen Teilen auswählen und diese der Bestellliste gleichzeitig hinzufügen. Durch das Lasso-Werkzeug sparen Sie Zeit, verringern Fehler und erhöhen die Verkäufe.

 

So fügen Sie der Bestellliste eine Teilegruppe hinzu:

  1. Navigieren Sie zu einem Bild.
  2. Klicken und ziehen Sie die Maus über die Teile und lasen Sie die Maustaste wieder los.

Wenn Sie die Teilegruppe auswählen, vergewissern Sie sich, dass Sie beim ersten Klick nicht auf eine Angabe klicken.

  1. Klicken Sie auf Der Bestellung hinzufügen.

Die Teilegruppe wird nun der Bestellliste hinzugefügt.

Das Fenster Teil auswählen wird automatisch angezeigt, wenn Teile ausgewählt werden müssen. Beispiel: LH und RH.

 

Sie können jederzeit auf den Bestell-Bildschirm zugreifen. Klicken Sie dazu am oberen Rand des Bildschirms auf Bestellungen.

Standardmäßig wird der Bestellliste die empfohlene Anzahl von jedem Teil hinzugefügt.Sie können die Anzahl im Feld Anz. bearbeiten.
