Fahrzeuge im Fahrzeugindex filtern

Der Fahrzeugindex zeigt alle Fahrzeuge in einer grafischen Sicht oder einer Textsicht an. Mithilfe des intuitiven Filters können Sie die Fahrzeugnamen während Ihrer Eingabe filtern.

 

So filtern Sie die Fahrzeuge:

  1. Geben Sie im Feld Filter die Filterkriterien ein. Geben Sie beispielsweise den Fahrzeugnamen und/oder das Baujahr ein.

Die Filterkriterien können aus einem einzelnen Wort oder aus mehreren Wörtern bestehen. Es werden nur die Fahrzeuge angezeigt, auf die die Filterkriterien zutreffen.

  1. Wählen Sie aus dem Index oder der Dropdown-Liste Filter ein Fahrzeug aus.
  2. Wählen Sie im Fenster Fahrzeugangaben aus den Dropdown-Listen die Hauptmerkmale aus.

Je mehr Hauptmerkmale eingegeben werden, desto genauer wird die Auswahl der Teile.
