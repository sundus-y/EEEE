Kostenberechnung ändern oder ausblenden

Über den Bestell-Bildschirm können Sie das Preisniveau ändern, das für Teile in der Bestellliste angewendet wird. Außerdem können Sie die Preise der Teile in der Bestellliste ausblenden. Das Ausblenden der Preise von Teilen ist sinnvoll, wenn Sie ein bestimmtes Teil direkt mit einem Kunden abklären.

Die Verfügbarkeit von Preisinformationen und Preisniveaus unterscheiden sich je nach Ihrem Hersteller.

 

So ändern Sie das Preisniveau:

  1. Zeigen Sie eine Bestellliste an.
  2. Wählen Sie aus der Dropdown-Liste Preis ein Preisniveau aus.

Das ausgewählte Preisniveau wird auf alle Teile in der Bestellliste angewendet.

 

So blenden Sie den Preis von Teilen aus:

  1. Zeigen Sie eine Bestellliste an.
  2. Wählen Sie aus der Dropdown-Liste Preis die Option Kostenberechnung ausblenden aus.

Die Spalten mit der Kostenberechnung werden in der Bestellliste ausgeblendet. Darüber hinaus wird die Kostenberechnung im Teiledaten-Raster sowie im Fenster Teiledaten ausgeblendet.

  1. Um die Kostenberechnung anzuzeigen, wählen Sie aus der Dropdown-Liste Preis ein Preisniveau aus.

 

Um Ihre Optionen für die Preisanzeige anzuzeigen, klicken Sie im Menü Einstellungen auf Allgemeine Einstellungen.
