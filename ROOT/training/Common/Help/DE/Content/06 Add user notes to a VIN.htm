Einer Fg.-Nr. Benutzernotizen hinzufügen

Microcat ermöglicht es Ihnen, einer Fg.-Nr. Benutzernotizen hinzuzufügen.

Fg.-Nr.-Notizen sind hilfreich, um einen Datensatz von einem bestimmten Fahrzeug zu erstellen. So können Sie beispielsweise eine Notiz hinzufügen, wenn ein Kundenfahrzeug mit Sportreifen ausgestattet ist.

 

So fügen Sie eine Fg.-Nr.-Notiz hinzu:

  1. Identifizieren Sie ein Fahrzeug anhand der Fg.-Nr.
  2. Klicken Sie im Fenster Fahrzeugangaben auf Fg.-Nr.-Notiz hinzufügen.
  3. Geben Sie den Titel und den Text der Notiz ein.
  4. Klicken Sie auf Absenden.

Der Fg.-Nr. wird das Symbol beigefügt. Bewegen Sie die Maus über das Symbol, um die Notiz anzuzeigen.

 

So bearbeiten Sie eine Fg.-Nr.-Notiz:

  1. Klicken Sie auf das Symbol , das der Fg.-Nr. beigefügt ist.
  2. Klicken Sie auf Bearbeiten.
  3. Bearbeiten Sie den Text.
  4. Klicken Sie auf Absenden.

 

So entfernen Sie eine Fg.-Nr.-Notiz:

  1. Klicken Sie auf das Symbol , das der Fg.-Nr. beigefügt ist.
  2. Klicken Sie auf Bearbeiten.
  3. Klicken Sie auf Entfernen.

 

Um eine Katalognotiz hinzuzufügen, klicken Sie im Fenster Fahrzeugangaben auf Katalognotiz hinzufügen.
