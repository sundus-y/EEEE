Katalogübergreifende Teilesuche ausführen

Microcat ermöglicht es Ihnen, katalogübergreifend nach einem bestimmten Teil zu suchen.

Die katalogübergreifende Suche ist hilfreich, um die entsprechenden Fahrzeuge auf eine bestimmte Teilenummer zu überprüfen. Sie haben auch die Möglichkeit, die Teilebeschreibung und den Preis zu überprüfen (sofern verfügbar).

 

So führen Sie eine katalogübergreifende Teilesuche aus:

  1. Geben Sie im Feld Suche nach eine Teilenummer ein.
  2. Vergewissern Sie sich, dass als Suchart die Option Teilenummer ausgewählt ist.
  3. Vergewissern Sie sich, dass das Kontrollkästchen Katalogübergreifende Suche aktiviert ist.
  4. Klicken Sie auf das Symbol oder drücken Sie auf der Tastatur auf Enter.

Die Suchergebnisse zeigen die Fahrzeuge an, auf die die Teilenummer zutrifft.

  1. Wählen Sie das Fahrzeug aus, um das Teil aufzurufen.
