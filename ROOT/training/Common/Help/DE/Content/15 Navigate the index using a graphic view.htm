Mithilfe einer grafischen Sicht durch den Index navigieren

Beim Grafikindex handelt es sich um einen Index mit Abbildungen der Baugruppen eines Fahrzeugs.

 

So navigieren Sie durch den Index mithilfe einer grafischen Sicht:

  1. Klicken Sie auf das Symbol , um eine grafische Sicht anzuzeigen.
  2. Klicken Sie im Index auf eine Hauptbaugruppe.
  3. Klicken Sie im Index auf eine Unterbaugruppe.
  4. Navigieren Sie weiter durch den Index zum Bild.
  5. Über das Bild können Sie Folgendes ausführen:
  • Doppelklicken Sie auf eine Angabe, um ein Teil der Bestellliste hinzuzufügen.
  • Klicken Sie auf eine Angabe, um das Fenster Teiledaten anzuzeigen.
  • Klicken Sie auf die Registerkarte Teile mit Abbildungen, um die Teileliste im Bild anzuzeigen.

 

Um das Fenster Teile mit Abbildungen anzuzeigen, klicken Sie in der Titelleiste auf das Symbol .
