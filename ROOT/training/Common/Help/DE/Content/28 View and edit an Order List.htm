Bestellliste anzeigen und bearbeiten

Im Bestell-Bildschirm können Sie eine Bestellliste anzeigen und bearbeiten.

Microcat ermöglicht es Ihnen, die Reihenfolge der Teile auf der Bestellliste zu ändern, um den Vorgaben eines Kunden zu entsprechen. Dadurch sparen Sie Zeit und bieten einen professionellen Service.

 

So entfernen Sie eine Zeile:

  1. Klicken Sie auf das Symbol in der Zeile, die Sie entfernen möchten.
  2. Klicken Sie auf Ja.

 

So kehren Sie zum Bild zurück:

  1. Klicken Sie auf das Symbol in der entsprechenden Zeile.

Es wird das Bild angezeigt, das das ausgewählte Teil enthält.

  1. Um zum Bestell-Bildschirm zurückzukehren, klicken Sie am oberen Rand des Bildschirms auf Bestellungen.

 

So bearbeiten Sie die Anzahl eines Teils:

  1. Klicken Sie auf das Feld Anz..
  2. Geben Sie im Feld Anz. die Anzahl ein.

 

Um die Reihenfolge der Teile auf der Bestellliste zu ändern, klicken Sie auf eine Zeile und ziehen Sie sie an die gewünschte Position.

Um die Reihenfolge der Liste der Teile auf der Bestellliste umzukehren, klicken Sie auf Ordnen.
