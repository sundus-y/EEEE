Informationen zu Nachfolgeteilen aufrufen

Wenn ein Teil durch andere Nachfolgeteile ersetzt wurde, wird im Teiledaten-Raster oder im Fenster Teiledaten eine Anzeige für Nachfolgeteile angezeigt. Eine Suche nach Nachfolgeteilen ist nützlich, um alte Lagerbestände zu räumen oder das Lager mit den aktuellsten Teileinformationen auf dem neuesten Stand zu halten.

Eine Anzeige für Nachfolgeteile besteht aus einem oder mehreren Zeichen und unterscheidet sich je nach Ihrem Hersteller. Die Anzeige wird im Teiledaten-Raster in der Spalte OSI, RECHN. oder S/S oder im entsprechenden Feld des Fensters Teiledaten angezeigt.

 

So rufen Sie Informationen zu Nachfolgeteilen auf:

  • Klicken Sie im Teiledaten-Raster auf die Anzeige für Nachfolgeteile.
  • Klicken Sie im Fenster Teiledaten auf die Anzeige für Nachfolgeteile.

 

So führen Sie eine Suche nach einem Nachfolgeteil durch:

  1. Klicken Sie auf die Anzeige für Nachfolgeteile.

Das Fenster Nachfolgeteil suchen wird angezeigt.

  1. Zeigen Sie die Informationen zum Nachfolgeteil an.

Es werden das Altteil und die aktuellsten Teileinformationen angezeigt.

  1. Aktivieren Sie das Kontrollkästchen, um das Teil der Bestellliste hinzuzufügen.
  2. Klicken Sie auf Schließen.
