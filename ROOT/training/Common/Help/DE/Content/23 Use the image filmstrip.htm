Filmstreifensicht verwenden

Mithilfe der Filmstreifensicht können Sie eine Vorschau der Baugruppen anzeigen, die sich vor oder nach der aktuellen Baugruppe befinden.

Wenn das Fahrzeug anhand der Fg.-Nr. identifizert wurde, wird in Rot oder Grün angezeigt, ob die Baugruppen geeignet sind.

 

So verwenden Sie die Filmstreifensicht:

  1. Navigieren Sie zu einem Bild.
  2. Klicken Sie auf eine beliebige Stelle im Bild, um den Bildbereich zu aktivieren.
  3. Drücken Sie auf der Tastatur die Tastenkombination Strg+Rechtspfeil.

Die Nummer und der Titel der Baugruppe wird in der Titelleiste der Filmstreifensicht angezeigt.

  1. Drücken Sie auf der Tastatur auf Linkspfeil oder Rechtspfeil, um eine Vorschau der Baugruppen anzuzeigen.
  2. Drücken Sie auf der Tastatur auf Enter, um eine Baugruppe auszuwählen.

Sie können Ihre Auswahl auch über Ihre Maus vornehmen.

 

Um die nicht zutreffenden Baugruppen anzuzeigen, klicken Sie im Menü Einstellungen auf Allgemeine Einstellungen und aktivieren Sie das Kontrollkästchen Nicht Zutreffendes anzeigen.
