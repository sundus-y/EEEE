Auf das Fenster Teiledaten zugreifen

Über das Fenster Teiledaten können Sie auf alle relevanten Teileinformationen zugreifen, beispielsweise auf die Teilebeschreibung, die Teilenummer und den Preis (sofern verfügbar).

Darüber hinaus gibt das Fenster Teiledaten an, ob Informationen zu Ersatzteilen verfügbar sind.

 

So greifen Sie auf das Fenster Teiledaten zu:

  • Klicken Sie im Teiledaten-Raster auf die Teilebeschreibung.
  • Klicken Sie im Bild auf die Angabe.

 

Vom Fenster Teiledaten aus können Sie Folgendes ausführen:

  • Zeigen Sie sämtliche Teileinformationen an, beispielsweise die Teilenummer oder das Gültigkeitsdatum von Teilen.
  • Klicken Sie auf die Anzeige von Nachfolgeteilen, um auf Informationen zu Ersatzteilen zuzugreifen.
  • Bearbeiten Sie im Feld Anz. die Anzahl eines Teils.
  • Klicken Sie auf Bestellen, um das Teil der Bestellliste hinzuzufügen.
  • Klicken Sie auf das Symbol , um das Teil in Ihr Händlersystem zu übertragen.
  • Klicken Sie auf die Symbole , um im Teiledaten-Raster zum nächsten oder zum vorherigen Teil zu springen.
