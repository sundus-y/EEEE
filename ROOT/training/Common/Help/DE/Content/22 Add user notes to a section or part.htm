Einer Baugruppe oder einem Teil Benutzernotizen hinzufügen

Baugruppen, Angaben oder Teilenummern können Benutzernotizen hinzugefügt werden. Benutzernotizen werden als gelbe Markierung angezeigt.

Durch das Hinzufügen von Notizen können Sie Verkaufsvorschläge oder nützliche Erinnerungen in Microcat anzeigen.

 

So fügen Sie eine Benutzernotiz hinzu:

  1. Wählen Sie den Notiztyp aus:
  • Baugruppennotiz: Klicken Sie links vom Bild auf das Symbol .
  • Teilenummer-Notiz: Klicken Sie im Teiledaten-Raster mit der rechten Maustaste auf das Teil und wählen Sie Notiz zur Teilenr. hinzufügen aus.
  • Angabe-Notiz: Klicken Sie im Bild auf die Angabe und wählen Sie Abruf-Angabe hinzufügen aus. Als Angabe kann der Herstellercode (beispielsweise PNC) stehen.
  1. Geben Sie den Titel und Text der Notiz ein.
  2. Klicken Sie auf Absenden.

Der Baugruppe, Teilenummer oder Angabe wird das Symbol hinzugefügt. Bewegen Sie die Maus über das Symbol, um die Notiz anzuzeigen.

 

So bearbeiten Sie eine Benutzernotiz:

  1. Klicken Sie auf das Symbol , das der Baugruppe, Teilenummer oder Angabe hinzugefügt ist.
  2. Klicken Sie auf Bearbeiten.
  3. Bearbeiten Sie den Text.
  4. Klicken Sie auf Absenden.

 

So entfernen Sie eine Benutzernotiz:

  1. Klicken Sie auf das Symbol , das der Baugruppe, Teilenummer oder Angabe hinzugefügt ist.
  2. Klicken Sie auf Bearbeiten.
  3. Klicken Sie auf Entfernen.
