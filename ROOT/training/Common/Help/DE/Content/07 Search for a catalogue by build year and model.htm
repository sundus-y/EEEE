Katalog anhand Baujahr und Modell suchen

In der Katalogsuche können Sie nach einem Fahrzeug suchen, indem Sie das Baujahr und Modell auswählen. Sie können dann die Suche verfeinern, indem Sie die Hauptmerkmale auswählen.

Die Katalogsuche ist hilfreich, wenn die Fg.-Nr. nicht bekannt ist. Je mehr Hauptmerkmale eingegeben werden, desto genauer wird die Auswahl der Teile.

 

So suchen Sie nach einem Katalog:

  1. Wählen Sie aus der Dropdown-Liste Baujahr das Baujahr aus.
  2. Wählen Sie aus der Dropdown-Liste Katalog den Fahrzeugkatalog aus.
  3. Wählen Sie im Fenster Fahrzeugangaben aus den Dropdown-Listen die Hauptmerkmale aus.

Suchen Sie nach Teilen über das Fenster Teilesuche oder Grafikindex.

 

Um die Fahrzeugmerkmale zu löschen, klicken Sie auf Zurücksetzen.
