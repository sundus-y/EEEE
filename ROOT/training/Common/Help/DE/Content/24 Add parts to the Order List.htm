Teile der Bestellliste hinzufügen

Wenn Sie die Teileangaben bestätigt haben, können Sie das Teil der Bestellliste hinzufügen.

Wenn Sie der Bestellliste Teile hinzufügen, können Sie dem Kunden ein Angebot oder eine Rechnung erstellen. Außerdem können Sie die Teile in Ihr Händlersystem übertragen.

 

Verwenden Sie die folgenden Optionen, um der Bestellliste Teile hinzuzufügen:

  • Doppelklicken Sie auf die Angabe im Bild.
  • Aktivieren Sie das Kontrollkästchen, das sich neben dem Teil im Teiledaten-Raster befindet.
  • Klicken Sie im Fenster Teiledaten auf Bestellen.
  • Klicken Sie mit der rechten Maustaste auf das Teil und wählen Sie Bestellen 1 oder Mehrere bestellen aus.

 

Die bestellten Teile werden in Violett angezeigt.

Standardmäßig wird der Bestellliste die empfohlene Anzahl von jedem Teil hinzugefügt.Sie können die Anzahl im Feld Anz. bearbeiten.
