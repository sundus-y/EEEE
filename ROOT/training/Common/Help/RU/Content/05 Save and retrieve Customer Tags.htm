Сохранять Метки клиента и переходить к ним

С помощью Меток клиента можно сохранить уникальное имя для отдельного автомобиля. Метки клиента удобно использовать для частых клиентов (например, клиентов парков).

 

Для сохранения Метки клиента:

  1. Найдите автомобиль с помощью VIN-кода.
  2. Введите Метку клиента в поле Метка клиента рядом с Панелью информации об автомобиле.
  3. Щелкните Сохранить.

 

Для перехода к Метке клиента:

  1. Введите Метку клиента в поле Метка клиента рядом с Панелью поиска автомобиля.
  2. Щелкните Поиск или нажмите Enter на клавиатуре.

При этом будет загружен каталог и отображена информация об автомобиле.

 

Чтобы перейти ко всему перечню Меток клиента, щелкните значок на Панели поиска автомобилей.
