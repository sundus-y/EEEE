Осуществлять поиск детали по всем каталогам

Microcat позволяет осуществлять поиск детали по всем каталогам.

Поиск по всем каталогам удобен для поиска автомобилей с определенным номером детали. Можно также найти описание детали и цену (если применимо).

 

Для поиска детали во всех каталогах:

  1. Введите номер детали в поле Поиск.
  2. Убедитесь, что в качестве типа поиска выбран Номер детали.
  3. Убедитесь, что стоит галочка напротив поля Перекрестный поиск по каталогам.
  4. Щелкните значок или нажмите Enter на клавиатуре.

В результатах поиска отобразятся автомобили с указанным номером детали.

  1. Выберите автомобиль, чтобы перейти к детали.
