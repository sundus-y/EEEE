Навигация в указателе посредством текстового интерфейса

Графический указатель можно также отобразить в текстовом виде.

 

Для навигации в указателе посредством текстового интерфейса:

  1. Щелкните значок для вызова текстового интерфейса.
  2. Щелкните основной раздел указателя.
  3. Щелкните подраздел указателя.
  4. Продолжите навигацию по указателю до перехода к изображению.
  5. На изображении можно:
  • Дважды щелкнуть выноску, чтобы добавить деталь в список заказов.
  • Щелкнуть выноску, чтобы открыть окно с информацией о детали.
  • Щелкнуть вкладку Иллюстрированные детали, чтобы просмотреть список деталей на изображении.

 

Чтобы перейти на уровень назад в указателе, щелкните значок .
