Поиск автомобиля по VIN-коду или по частичному VIN-коду

Поиск по VIN-коду является самым надежным способом идентификации автомобиля и обеспечивает наиболее точный выбор деталей.

Виды поиска различаются в зависимости от изготовителя. Например, возможен поиск по регистрационному номеру, номеру модели и номеру рамы.

 

Для поиска по VIN-коду:

  1. Введите VIN-код в поле Поиск.

Можно также выбрать VIN-код из раскрывающегося списка последних искомых VIN-кодов.

  1. Щелкните Поиск или нажмите Enter на клавиатуре.

При этом будет загружен каталог и отображена информация об автомобиле.

Для поиска деталей воспользуйтесь Панелью поиска деталей или Графическим указателем.

 

Для поиска по частичному VIN-коду:

  1. Введите последние 8 символов VIN-кода в поле Поиск.

Счетчик символов рядом с полем VIN-кода будет показывать число введенных символов.

  1. Щелкните Поиск или нажмите Enter на клавиатуре.
  2. Если отобразятся несколько VIN-кодов, выберите требуемый.

При этом будет загружен каталог и отображена информация об автомобиле.

Для поиска деталей воспользуйтесь Панелью поиска деталей или Графическим указателем.

 

Чтобы распечатать или отправить по эл. почте информацию о текущем автомобиле, выберите Печать/Отправить эл. почту.

Чтобы отобразить Панель поиска автомобилей, щелкните значок в строке заголовка.
