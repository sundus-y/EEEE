Поиск деталей по коду детали изготовителя:

Возможен поиск деталей по ряду типов поиска, в том числе по номеру детали или коду ее изготовителя.

Виды поиска различаются в зависимости от изготовителя. Например, по основному номеру, FINIS-коду, PNC или коду товара.

 

Для поиска деталей по коду детали изготовителя:

  1. Введите код детали изготовителя в поле Поиск.

Например, по основному номеру, FINIS-коду, PNC или коду товара.

  1. Выберите соответствующий тип поиска.
  2. Щелкните значок или нажмите Enter на клавиатуре.

Отобразятся результаты поиска.

  1. В результатах поиска можно:
  • Отметить галочкой, чтобы добавить деталь в список заказов.
  • Щелкнуть значок для просмотра детали на изображении.
  • Щелкнуть описание детали, чтобы открыть окно с информацией о детали.

 

Чтобы вернуться к результатам поиска из изображения, щелкните вкладку Результаты поиска.
