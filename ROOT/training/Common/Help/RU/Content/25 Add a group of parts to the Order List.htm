Добавление группы деталей в список заказов

Чтобы выбрать и добавить сразу группу из нескольких деталей в список заказов, можно воспользоваться инструментом лассо. Благодаря инструменту лассо экономится время, сокращается количество ошибок и повышаются продажи.

 

Чтобы добавить группу деталей в список заказов:

  1. Перейдите к изображению.
  2. Щелкните и захватите указателем мыши детали, после чего отпустите его.

При выделении группы деталей следите за тем, чтобы первый щелчок не пришелся на выноску.

  1. Выберите Добавить в заказ.

Группа деталей будет добавлена в список заказов.

Если требуется выбрать между деталями, автоматически отобразится окно списка обнаруженных деталей, например LH и RH.

 

Чтобы открыть окно заказов, щелкните Заказы в верхней части экрана.

В список заказов по умолчанию добавляется рекомендованное количество каждой детали. Количество можно изменить в поле Кол-во.
