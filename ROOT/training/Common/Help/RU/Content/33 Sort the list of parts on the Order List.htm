Сортировать список деталей в списке заказов

Можно сортировать список деталей в списке заказов в обратном порядке для удовлетворения пожеланий клиента или для соответствия параметров отображения, используемых другими системами.

 

Чтобы сортировать список деталей в списке заказов:

  1. Добавьте детали в список заказов.
  2. Щелкните Сортировать.

Список деталей в списке заказов будет отображаться в обратном порядке.

  1. При необходимости продолжайте добавлять детали в список заказов.

Детали будут добавляться в верхнюю или нижнюю часть списка заказов.

 

Если отображается стрелка вверх, детали будут добавляться в нижнюю часть списка, а если стрелка вниз — в верхнюю часть списка.
