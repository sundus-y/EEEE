Добавлять примечания пользователя к VIN-коду

Microcat позволяет добавлять примечания пользователя к VIN-коду.

Примечания к VIN-коду удобно использовать для создания записи об определенном автомобиле. Например, примечание можно добавить при установке спортивных колес на автомобиль клиента.

 

Для добавления примечания к VIN-коду:

  1. Найдите автомобиль с помощью VIN-кода.
  2. Щелкните Добавить примечание к VIN-коду на Панели информации об автомобиле.
  3. Введите наименование и текст примечания.
  4. Щелкните Отправить.

К VIN-коду будет прикреплен значок . Чтобы вывести на экран примечание, наведите указатель мыши на значок.

 

Чтобы изменить примечание к VIN-коду:

  1. Щелкните значок , который прикреплен к VIN-коду.
  2. Щелкните Изменить.
  3. Внесите изменения в текст.
  4. Щелкните Отправить.

 

Чтобы удалить примечание к VIN-коду:

  1. Щелкните значок , который прикреплен к VIN-коду.
  2. Щелкните Изменить.
  3. Щелкните Удалить.

 

Чтобы добавить примечание к каталогу, выберите Добавить примечание к каталогу на Панели информации об автомобиле.
