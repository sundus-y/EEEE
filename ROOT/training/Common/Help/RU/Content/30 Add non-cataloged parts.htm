Добавление деталей не из каталога

Microcat позволяет легко добавлять в список заказов элементы не из каталога.

Окно ввода деталей в свободной форме позволяет включить в список заказов любые элементы, в том числе расходные материалы для мастерских. Это необходимо при создании ценового предложения или счета для клиента.

 

Чтобы добавить деталь не из каталога:

  1. Щелкните Ввод деталей в свободной форме в верхней части списка заказов.
  2. Введите в соответствующие поля номер, описание, количество и цену для детали.
  3. Выберите Добавить в заказ.

Деталь не из каталога будет добавлена в список заказов.

 

Чтобы передать детали не из каталога в DMS, в системе должны быть зарегистрированы номера деталей.
