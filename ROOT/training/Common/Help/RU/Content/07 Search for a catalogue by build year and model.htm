Осуществлять поиск по каталогу с учетом года выпуска и модели

Поиск по каталогу позволяет находить автомобиль, указав для этого год выпуска и модель. Затем можно сузить область поиска, выбрав основные характеристики.

Поиск по каталогу удобен, если неизвестен VIN-код. Чем больше указано основных характеристик, тем более точным будет выбор деталей.

 

Для поиска по каталогу:

  1. В раскрывающемся списке Год выберите год выпуска.
  2. В раскрывающемся списке Каталог выберите автомобильный каталог.
  3. Выберите основные характеристики из раскрывающихся списков на Панели информации об автомобиле.

Для поиска деталей воспользуйтесь Панелью поиска деталей или Графическим указателем.

 

Чтобы очистить характеристики автомобиля, щелкните Сброс.
