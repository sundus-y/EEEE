Поиск деталей по описанию

Возможен поиск деталей по ряду типов поиска, в том числе по описанию детали. Наглядный фильтр позволяет сортировать результаты поиска по мере ввода информации.

 

Для поиска деталей по описанию:

  1. Введите описание в поле Поиск.

Введенное описание должно соответствовать описанию изготовителя в каталоге.

  1. Убедитесь, что в качестве типа поиска выбрано Описание.
  2. Щелкните значок или нажмите Enter на клавиатуре.

Отобразятся результаты поиска.

  1. Чтобы ограничить результаты поиска, введите критерии фильтра в поле фильтра.

В качестве критериев фильтра может выступать как одно, так и несколько слов. Отобразятся только результаты поиска, отвечающие критериям фильтра.

  1. В результатах поиска можно:
  • Отметить галочкой, чтобы добавить деталь в список заказов.
  • Щелкнуть значок для просмотра детали на изображении.
  • Щелкнуть описание детали, чтобы открыть окно с информацией о детали.

 

Для отображения или скрытия информации об автомобиле используйте значок .
