向车辆识别号添加用户备注

Microcat 可让您向车辆识别号添加用户备注。

车辆识别号备注对于创建特定车辆记录十分有用。 例如，当运动式车轮与某客户的车辆匹配时添加备注。

 

若要添加车辆识别号备注：

  1. 按车辆识别号识别车辆。
  2. 单击车辆信息面板上的添加车辆识别号备注。
  3. 键入备注标题和文本。
  4. 单击提交。

图标将附加到车辆识别号。 将鼠标移到图标上，以显示备注。

 

若要编辑车辆识别号备注：

  1. 单击附加到车辆识别号的 图标。
  2. 单击编辑。
  3. 编辑文本。
  4. 单击提交。

 

若要删除车辆识别号备注：

  1. 单击附加到车辆识别号的 图标。
  2. 单击编辑。
  3. 单击删除。

 

若要添加目录备注，单击车辆信息面板上的添加目录备注。
