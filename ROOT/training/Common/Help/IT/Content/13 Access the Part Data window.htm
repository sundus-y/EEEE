Accedere alla finestra Dati ricambi

La finestra Dati ricambi vi permette di accedere a tutte le informazioni sui ricambi, quali la descrizione del ricambio, il numero del ricambio e il prezzo (se disponibile).

La finestra Dati ricambi indicherà inoltre se sono disponibili le informazioni sul ricambio sostitutivo.

 

Per accedere alla finestra Dati ricambi:

  • Fate clic sulla descrizione del ricambio nella Griglia dati ricambi.
  • Fate clic sul callout nell'immagine.

 

Dalla finestra Dati ricambi potete:

  • Visualizzare tutte le informazioni sui ricambi quali il numero ricambio o le date di validità del ricambio.
  • Fare clic sull'indicatore del ricambio sostitutivo per accedere alle informazioni sul ricambio sostitutivo.
  • Modificare la quantità di un ricambio nel riquadro Quant..
  • Fare clic su Ordine per aggiungere un ricambio all'Elenco ordini.
  • Fare clic sull'icona per trasferire il ricambio al DMS.
  • Fare clic sulle icone per spostarvi dal ricambio precedente a quello successivo nella Griglia dati ricambi.
