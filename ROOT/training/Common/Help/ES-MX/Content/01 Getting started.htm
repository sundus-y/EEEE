Para comenzar

Microcat es un sistema de venta de refacciones de última generación.

 

Entre los beneficios de usar Microcat están:

  • La capacidad de proporcionar los datos más recientes de las refacciones
  • Una interfaz de usuario, navegación y flujo de trabajo más eficiente
  • Características intuitivas para identificar el vehículo y buscar refacciones
  • Capacidad aumentada para integrarse con su Sistema de gestión de concesionario

 

Hay tres zonas de trabajo principales en Microcat:

  • Página Inicio
  • Pantalla Refacciones
  • Pantalla Pedidos
