Pantalla Refacciones

La pantalla Refacciones se utiliza para localizar refacciones, agregar refacciones a la Lista de pedidos y transferir refacciones a su Sistema de gestión de concesionario (DMS).

Cuando identifique un vehículo, se mostrará automáticamente la pantalla Refacciones. Para acceder a la pantalla de Refacciones, pulse Refacciones en la parte superior de la pantalla.

 

Desde la página de Refacciones puede:

  • Buscar refacciones utilizando una gama de tipos de búsqueda
  • Navegar por el índice usando una vista gráfica o en texto
  • Usar la herramienta de navegación rápida
  • Mostrar las imágenes de diferentes maneras
  • Trabajar con la Cuadrícula de datos de refacciones
  • Agregar notas de usuario a una sección o una refacción
  • Agregar refacciones a la Lista de pedidos
  • Transferir una refacción a su DMS

 

Presione F11 en el teclado para mostrar la vista de pantalla completa de Microcat.
