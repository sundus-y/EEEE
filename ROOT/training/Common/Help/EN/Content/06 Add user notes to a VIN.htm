Add user notes to a VIN

Microcat allows you to add user notes to a VIN.

VIN notes are useful to create a record of a particular vehicle. For example, add a note when sports wheels are fitted to a customer’s vehicle.

 

To add a VIN note:

  1. Identify a vehicle by VIN.
  2. Click Add VIN Note in the Vehicle Information panel.
  3. Type your note title and text.
  4. Click Submit.

A icon will be attached to the VIN. Move the mouse over the icon to display the note.

 

To edit a VIN note:

  1. Click the icon that is attached to the VIN.
  2. Click Edit.
  3. Edit the text.
  4. Click Submit.

 

To delete a VIN note:

  1. Click the icon that is attached to the VIN.
  2. Click Edit.
  3. Click Delete.

 

To add a catalogue note, click Add Catalogue Note in the Vehicle Information panel.
