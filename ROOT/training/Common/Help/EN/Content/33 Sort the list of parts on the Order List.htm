Sort the list of parts on the Order List

You can reverse the list of parts on the Order List to meet a customer's preferences or to match the display used by other systems.

 

To sort the list of parts on the Order List:

  1. Add parts to the Order List.
  2. Click Sort.

The list of parts on the Order List will be reversed.

  1. Continue to add parts to the Order List, as required.

The parts will be added to the top or bottom of the Order List.

 

If an up arrow is displayed, parts will be added to the bottom of the list and if a down arrow is displayed, parts will be added to the top of the list.
