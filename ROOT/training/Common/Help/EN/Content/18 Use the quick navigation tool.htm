Use the quick navigation tool

The quick navigation tool is located at the top of the screen and allows you to enter shortcuts to jump between catalogues or sections.

The quick navigation tool is useful for both new and experienced users.

 

Shortcut

Jump to

<.>+<catalogue code>+<Enter>

Catalogue using a catalogue code

<.>+<last 8 characters of VIN>+<.>+<Enter>

Catalogue using a VIN

<section number>+<Enter>

Section inside a catalogue

 

To use the quick navigation tool:

  1. Type a shortcut into the quick navigation box.
  2. Click the icon or press Enter on the keyboard.

The catalogue, section or search results will be displayed.

  1. If multiple results are displayed, select the option you require.

 

Move the mouse over the icon to display a complete list of shortcuts.
