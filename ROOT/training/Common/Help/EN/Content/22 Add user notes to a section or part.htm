Add user notes to a section or part

User notes can be added to sections, callouts or part numbers. User notes are displayed as a yellow flag.

Adding notes allows you to display sales suggestions or useful reminders in Microcat.

 

To add a user note:

  1. Select the type of note:
  • Section note: Click the icon on the left of the image.
  • Part number note: Right-click the part in the Parts Data Grid and select Add Part Number Note.
  • Callout note: Right-click the callout in the image and select Add Callout Note. The callout may be referred to as the manufacturer part code (e.g. PNC).
  1. Type the note title and text.
  2. Click Submit.

A icon will be attached to the section, part number or callout. Move the mouse over the icon to display the note.

 

To edit a user note:

  1. Click the icon that is attached to the section, part number or callout.
  2. Click Edit.
  3. Edit the text.
  4. Click Submit.

 

To delete a user note:

  1. Click the icon that is attached to the section, part number or callout.
  2. Click Edit.
  3. Click Delete.
