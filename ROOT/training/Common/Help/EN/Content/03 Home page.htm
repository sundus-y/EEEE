Home page

The Home page is the first page you see when you log on to Microcat.

 

From the Home page you can:

  • Search for a vehicle by VIN or by partial VIN
  • Save and retrieve Customer Tags
  • Add user notes to a VIN
  • Search for a catalogue by build year and model
  • Filter the vehicles in the Vehicle Index
  • Search for a part across all catalogues

 

To return to the Home page at any time, click Home.
