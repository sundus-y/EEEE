Search for parts by description

You can search for parts using a range of search types including the part description. The intuitive filter allows you to filter the search results as you type.

 

To search for parts by description:

  1. Type a description into the Search for box.

The description you enter must be the same as the description used by the manufacturer in the catalogue.

  1. Confirm Description is selected as the search type.
  2. Click the icon or press Enter on the keyboard.

The search results will be displayed.

  1. If you want to reduce your search results, type a filter criteria into the filter box.

The filter criteria can be one or multiple words. Only the search results that contain the filter criteria will be displayed.

  1. From the search results, you can:
  • Click the checkbox to add the part to the Order List.
  • Click the icon to view the part in the image.
  • Click the part description to display the Part Data window.

 

To show or hide vehicle information, click the icon.
