Add parts to the Order List

Once you have confirmed the part details, the part can be added to the Order List.

Adding parts to the Order List allows you to provide a quote or invoice to a customer. In addition, you can transfer parts to your Dealer Management System.

 

To add parts to the Order List, use the following options:

  • Double-click the callout in the image.
  • Click the checkbox next to the part in the Parts Data Grid.
  • Click Order in the Part Data window.
  • Right-click on the part and select Order 1 or Order Many.

 

Parts that have been ordered will be displayed in purple.

By default the recommended quantity of each part is added to the Order List. You can edit the quantity in the Qty box.
