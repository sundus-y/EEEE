Filter the vehicles in the Vehicle Index

The Vehicle Index displays all the vehicles using a graphic or text view. The intuitive filter allows you to filter the vehicle names as you type.

 

To filter the vehicles:

  1. Type the filter criteria into the filter box. For example, the vehicle name and/or year.

The filter criteria can be one or multiple words. Only the vehicles that contain the filter criteria will be displayed.

  1. Select a vehicle from the index or from the filter drop-down list.
  2. Select the major attributes from the drop-down lists in the Vehicle Information panel.

The more major attributes that are entered the more accurate the selection of parts.
