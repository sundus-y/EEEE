Search for a part across all catalogues

Microcat allows you to search for a part across all catalogues.

The cross catalogue search is useful to check the applicable vehicles for a particular part number. You can also check the part description and price (if available).

 

To search for a part across all catalogues:

  1. Type a part number into the Search for box.
  2. Confirm Part Number is selected as the search type.
  3. Confirm the Cross Catalogue checkbox is selected.
  4. Click the icon or press Enter on the keyboard.

The search results will display the applicable vehicles for the part number.

  1. Select the vehicle to go to the part.
