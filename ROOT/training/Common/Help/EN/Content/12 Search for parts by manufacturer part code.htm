Search for parts by manufacturer part code

You can search for parts using a range of search types including the part number or the manufacturer part code.

The search types will vary based on your manufacturer. For example, basic number, FINIS code, PNC or commodity code.

 

To search for parts by manufacturer part code:

  1. Type the manufacturer part code into the Search for box.

For example, basic number, FINIS Code, PNC or commodity code.

  1. Select the corresponding search type.
  2. Click the icon or press Enter on the keyboard.

The search results will be displayed.

  1. From the search results, you can:
  • Click the checkbox to add the part to the Order List.
  • Click the icon to view the part in the image.
  • Click the part description to display the Part Data window.

 

To return to the search results from the image, click the Search Results tab.
