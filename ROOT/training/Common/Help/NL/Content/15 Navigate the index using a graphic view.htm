Door de index navigeren met behulp van de grafische weergave

De Grafische index is een index met afbeeldingen van de secties van een voertuig.

 

Door de index navigeren met behulp van de grafische weergave:

  1. Klik op het pictogram voor de grafische weergave.
  2. Klik vanuit de index op een hoofdsectie.
  3. Klik vanuit de index op een subsectie.
  4. Blader verder door de index naar de afbeelding.
  5. Vanuit de afbeelding kunt u:
  • Dubbelklikken op een referentienummer om een onderdeel aan de bestellijst toe te voegen.
  • Op een referentienummer klikken om het venster Onderdeelgegevens weer te geven.
  • Op het tabblad Geïlllustreerde onderdelen klikken om de onderdelenlijst in de afbeelding weer te geven.

 

Op het pictogram op de titelbalk klikken om het venster Onderdelen met afbeelding weer te geven.
