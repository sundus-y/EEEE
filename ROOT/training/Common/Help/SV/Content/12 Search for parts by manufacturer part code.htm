Söka efter reservdelar via tillverkarkod

Du kan söka efter reservdelar via en rad olika söktyper inklusive reservdelsnummer eller reservdelskod.

Söktyperna varierar beroende på din tillverkare. Till exempel grundnummer, finisnummer, PNC eller varukod.

 

För att söka efter reservdelar via tillverkarkod:

  1. Ange tillverkarkoden i rutan Sök efter.

Till exempel grundnummer, finisnummer, PNC eller varukod.

  1. Välj motsvarande söktyp.
  2. Klicka på -ikonen eller tryck på Enter på tangentbordet.

Sökresultaten visas.

  1. Från sökresultaten kan du göra följande:
  • Klicka på kryssrutan för att lägga till reservdelen i orderlistan.
  • Klicka på -ikonen för att visa reservdelen på bilden.
  • Klicka på reservdelsbeskrivningen för att visa fönstret Reservdelsdata.

 

För att återgå till sökresultaten från bilden, klicka på fliken Sökresultat.
