Söka efter fordon på chassinummer eller del av chassinummer

Sökning på chassinummer är den mest pålitliga metoden för att identifiera ett fordon och ger det mest tillförlitliga reservdelsurvalet.

Söktyperna varierar beroende på din tillverkare. Till exempel registreringsnummer, modellnummer och ramnummer.

 

För att söka med chassinummer:

  1. Ange chassinumret i rutan Sök på.

Du kan även välja ett chassinummer från rullgardinsmenyn med tidigare chassinummersökningar.

  1. Klicka på Sök eller tryck på Enter på tangentbordet.

Katalogen läses in och fordonsinformationen visas.

Använd fönstret Reservdelssökning eller Grafiskt index för att hitta reservdelar.

 

För att söka på del av chassinummer:

  1. Ange chassinumrets sista 8 tecken i rutan Sök på.

Teckenräknaren bredvid chassinummerrutan visar antalet tecken som har angivits.

  1. Klicka på Sök eller tryck på Enter på tangentbordet.
  2. Välj önskat chassinummer om fler än ett chassinummer visas.

Katalogen läses in och fordonsinformationen visas.

Använd fönstret Reservdelssökning eller Grafiskt index för att hitta reservdelar.

 

För att skriva ut fordonsinformation eller skicka via e-post, klicka på Utskrift/E-post.

För att visa fönstret Fordonssökning, klicka på -ikonen på namnlisten.
