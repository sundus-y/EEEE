Lägga till användaranteckningar till ett avsnitt eller en reservdel

Man kan lägga till användaranteckningar till avsnitt, kodbeteckningar eller reservdelsnummer. Användaranteckningar visas som en gul flagga.

Genom att lägga till anteckningar kan du visa försäljningsförslag eller påminnelser i Microcat.

 

För att lägga till en användaranteckning:

  1. Välj typ av anteckning:
  • Avsnittsanteckning: Klicka på -ikonen till vänster om bilden.
  • Anteckning om reservdelsnummer: Högerklicka på reservdelen i reservdelstabellen och välj Lägg till anteckning om reservdelsnummer.
  • Kodbeteckningsanteckning: Högerklicka på kodbeteckningen på bilden och välj Lägg till kodbeteckningsanteckning. Kodbeteckningen kan eventuellt kallas tillverkarkod (t.ex. PNC).
  1. Ange titel på anteckningen och skriv in informationen.
  2. Klicka på Skicka.

En -ikon bifogas avsnittet, reservdelsnumret eller kodbeteckningen. Rör musen över ikonen för att visa anteckningen.

 

För att redigera en användaranteckning:

  1. Klicka på -ikonen som bifogats avsnittet, reservdelsnumret eller kodbeteckningen.
  2. Klicka på Redigera.
  3. Redigera texten.
  4. Klicka på Skicka.

 

För att radera en användaranteckning:

  1. Klicka på -ikonen som bifogats avsnittet, reservdelsnumret eller kodbeteckningen.
  2. Klicka på Redigera.
  3. Klicka på Radera.
