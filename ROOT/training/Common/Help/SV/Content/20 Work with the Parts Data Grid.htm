Arbeta med reservdelstabellen

Reservdelstabellen visar reservdelsinformation i tabellform under bilden. Du kan lätt växla mellan flikarna Illustrerade reservdelar och Sökresultat.

Detta kan vara till hjälp då du kan återgå till de senaste sökresultaten genom att klicka på fliken Sökresultat.

 

Reservdelstabellen består av två flikar:

  • Illustrerade reservdelar: Visar en lista över illustrerade avsnitt och reservdelar. Klicka på kryssrutan för att lägga till reservdelen i orderlistan. Klicka på beskrivningen för att visa fönstret Reservdelsdata.
  • Sökresultat: Visar sökresultaten efter en sökning. Klicka på -ikonen för att visa reservdelen på bilden.

 

Använd följande alternativ för att justera reservdelstabellen:

  • Klicka på -ikonerna på namnlisten för att visa eller dölja fönstret.
  • Dubbelklicka på namnlisten för att visa eller dölja fönstret.
  • Klicka och dra pilen med två spetsar uppåt eller nedåt för att justera fönstrets höjd.

 

För att justera spaltvisning:

  • Klicka på spaltrubriken för att sortera informationen i spalten.
  • Klicka och dra spaltrubriken till en ny position i reservdelstabellen.
  • Klicka och dra den högra sidan av spaltrubriken till höger eller vänster för att justera spaltens bredd.
