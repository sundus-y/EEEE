Vy med delad skärm

Med Microcat kan man välja en vy med delad skärm när man väljer reservdelar. På vyn med delad skärm visas reservdelstabellen till vänster och bilden till höger.

På så sätt kan du se all reservdelsinformation som tillhör en viss bild. Om fordonet dessutom har identifierats via chassinummer, kan du visa reservdelarnas tillämplighet.

 

För att öppna vy med delad skärm:

  1. Navigera till en bild.
  2. Klicka på -ikonen för att öppna vy med delad skärm.

Reservdelstabellen visas till vänster och bilden visas till höger.

  1. Visa tillämplighet för varje reservdel:
  • Grön: passande
  • Blå: ej-original
  • Röd: passar ej

För att visa icke-tillämpliga reservdelar, klicka på Allmänna inställningar från menyn Inställningar och markera kryssrutan Visa även ej tillämpliga.

  1. Välj en reservdel i reservdelstabellen eller från bilden.

 

För att visa bilden ovanför reservdelstabellen, klicka på -ikonen.
