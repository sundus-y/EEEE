Söka efter en katalog med hjälp av årsmodell och modell

Med katalogsökningen kan du söka efter ett fordon genom att välja årsmodell och modell. Därefter kan du förfina sökningen genom att välja huvudkomponenter.

Katalogsökningen är praktisk om man inte känner till chassinumret. Ju fler huvudkomponenter som anges desto mer tillförlitligt blir reservdelsurvalet.

 

För katalogsökning:

  1. Välj årsmodellen från rullgardinsmenyn År.
  2. Välj önskad fordonskatalog från rullgardinsmenyn Katalog.
  3. Välj huvudegenskaper från rullgardinsmenyn i fönstret Fordonsinformation.

Använd fönstret Reservdelssökning eller Grafiskt index för att hitta reservdelar.

 

För att rensa fordonsegenskaper, klicka på Återställ.
