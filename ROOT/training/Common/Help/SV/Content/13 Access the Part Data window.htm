Öppna fönstret Reservdelsdata

På fönstret Reservdelsdata kan du öppna all relevant reservdelsinformation som till exempel reservdelsnummer och pris (om tillgängligt).

Om information om ersättningsdel är tillgänglig, visas även denna på fönstret Reservdelsdata.

 

För att öppna fönstret Reservdelsdata:

  • Klicka på reservdelsbeskrivningen i reservdelstabellen.
  • Klicka på kodbeteckningen på bilden.

 

Från fönstret Reservdelsdata kan du göra följande:

  • Visa all reservdelsinformation som till exempel reservdelsnummer eller giltighetsdatum för reservdelar.
  • Klicka på ersättningsindikatorn för att öppna information om ersättningsdel.
  • Ändra antalet reservdelar i rutan Ant..
  • Klicka på Beställa för att lägga till reservdelen i orderlistan.
  • Klicka på -ikonen för att överföra reservdelen till DMS.
  • Klicka på -ikonerna för att gå till nästa eller föregående del reservdelstabellen.
