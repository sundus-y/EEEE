Söka efter reservdelar med beskrivning

Du kan söka efter reservdelar med en rad olika söktyper inklusive reservdelsbeskrivningen. Det intuitiva filtret gör att du kan filtrera sökresultaten samtidigt som du skriver.

 

Söka efter reservdelar med beskrivning:

  1. Ange en beskrivning i rutan Sök efter.

Beskrivningen du anger måste överensstämma med beskrivningen som tillverkaren använder i katalogen.

  1. Kontrollera att Beskrivning har valts som typ av sökning.
  2. Klicka på -ikonen eller tryck på Enter på tangentbordet.

Sökresultaten visas.

  1. Ange ett filterkriterium i filterrutan om du vill begränsa sökresultatet.

Filterkriterierna kan vara ett eller flera ord. Endast sökresultaten som innehåller filterkriterierna visas.

  1. Från sökresultaten kan du göra följande:
  • Klicka på kryssrutan för att lägga till reservdelen i orderlistan.
  • Klicka på -ikonen för att visa reservdelen på bilden.
  • Klicka på reservdelsbeskrivningen för att visa fönstret Reservdelsdata.

 

För att visa eller dölja fordonsinformation, klicka på -ikonen.
