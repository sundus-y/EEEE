Visa och redigera orderlistor

På orderfönstret kan du visa och redigera orderlistor.

Med Microcat kan du ändra ordningsföljden på reservdelar på orderlistan för att bemöta kundernas behov. Du sparar tid och erbjuder en professionell tjänst.

 

För att ta bort rad:

  1. Klicka på -ikonen på raden som du ta bort.
  2. Klicka på Ja.

 

För att återgå till bilden:

  1. Klicka på -ikonen på den aktuella raden.

Bilden som innehåller den valda reservdelen visas.

  1. För att återgå till orderfönstret, klicka på Beställningar högst upp i fönstret.

 

För att redigera reservdelens antal:

  1. Klicka på rutan Ant..
  2. Ange antal i rutan Ant..

 

För att ändra ordningsföljden på reservdelarna i orderlistan, klicka och dra en rad till önskad position.

För att ändra ordningsföljden på listan över reservdelar i orderlistan, klicka på Sortera.
