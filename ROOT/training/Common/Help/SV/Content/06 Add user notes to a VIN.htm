Ange användaranteckningar för ett chassinummer

Med Microcat kan du ange användaranteckningar för ett chassinummer.

Chassinummeranteckningar är praktiska när man vill skapa en fil för ett visst fordon. Till exempel kan man lägga till en anteckning om man monterar sporthjul på fordonet.

 

För att lägga till en chassinummeranteckning:

  1. Sök fordon på chassinummer.
  2. Klicka på Lägg till en anteckning till chassinumret i fönstret Fordonsinformation.
  3. Ange titel på anteckningen och skriv in informationen.
  4. Klicka på Skicka.

En -ikon kommer att bifogas chassinumret. Rör musen över ikonen för att visa anteckningen.

 

För att redigera en chassinummeranteckning:

  1. Klicka på -ikonen som är bifogad chassinumret.
  2. Klicka på Redigera.
  3. Redigera texten.
  4. Klicka på Skicka.

 

För att radera en chassinummeranteckning:

  1. Klicka på -ikonen som är bifogad chassinumret.
  2. Klicka på Redigera.
  3. Klicka på Radera.

 

För att lägga till en kataloganteckning, klicka på Lägg till en kataloganteckning i fönstret Fordonsinformation.
