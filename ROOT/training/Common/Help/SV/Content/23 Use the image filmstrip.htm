Miniatyrbilder

Med miniatyrbilder kan man förhandsgranska avsnitt före eller efter det avsnitt som för tillfället visas.

Om fordonet har identifierats via chassinummer visas avsnittens tillämplighet i rött eller grönt.

 

För att använda miniatyrbilder:

  1. Navigera till en bild.
  2. Klicka varsomhelst på bilden för att aktivera bildytan.
  3. Tryck Ctrl+Högerpil på tangentbordet.

Avsnittsnummer och titel visas på miniatyrbildens namnlist.

  1. Tryck på Vänster- eller Högerpil på tangentbordet för att förhandsgranska avsnitten.
  2. Tryck på Enter på tangentbordet för att välja ett avsnitt.

Du kan även använda musen för att välja ett avsnitt.

 

För att visa icke-tillämpliga avsnitt, klicka på Allmänna inställningar från menyn Inställningar och markera kryssrutan Visa även ej tillämpliga.
