Navigera i indexet med grafisk vy

Grafiskt index är ett bildindex över fordonets olika avsnitt.

 

För att navigera i indexet med grafisk vy:

  1. Klicka på -ikonen för att visa grafisk vy.
  2. Klicka på ett huvudavsnitt i indexet.
  3. Klicka på ett underavsnitt i indexet.
  4. Fortsätt att navigera genom indexet till bilden.
  5. Från bilden kan du:
  • Dubbelklicka på en kodbeteckning för att lägga till en reservdel i orderlistan.
  • Klicka på en kodbeteckning för att visa fönstret Reservdelsdata.
  • Klicka på fliken Illustrerade reservdelar för att visa en lista över reservdelarna på bilden.

 

För att visa fönstret Reservdelar, klicka på -ikonen på namnlisten.
