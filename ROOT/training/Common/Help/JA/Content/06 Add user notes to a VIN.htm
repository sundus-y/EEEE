ユーザーメモの VIN への追加

Microcat ではユーザーメモを VIN に追加することができます。

VIN メモは、特定の車両の記録作成に役立ちます。 例：顧客車両にスポーツタイヤが装着されている場合にメモを追加する。

 

VIN メモを追加する：

  1. VIN で車両を特定します。
  2. 車両情報パネル のVIN メモの追加をクリックします。
  3. メモのタイトルとテキストを入力します。
  4. 提出をクリックします。

VIN に  のアイコンが付けられます。 メモを表示するには、アイコンの上にマウスを合わせます。

 

VIN メモを編集する：

  1. VIN に付けられた  のアイコンをクリックします。
  2. 編集をクリックします。
  3. テキストを編集します。
  4. 提出をクリックします。

 

VIN メモを削除する：

  1. VIN に付けられた  のアイコンをクリックします。
  2. 編集をクリックします。
  3. 削除をクリックします。

 

カタログメモを追加するには、車両情報パネルのカタログメモの追加をクリックします。
