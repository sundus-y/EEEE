テキストビューを使用したインデックスのナビゲート

イラストインデックスをテキストビューとして表示することもできます。

 

テキストビューを使用してインデックスをナビゲートする：

  1. のアイコンをクリックして、テキストビューを表示します。
  2. インデックスから主要セクションをクリックします。
  3. インデックスからマイナーセクションをクリックします。
  4. インデックスから画像までナビゲートを続けます。
  5. 画像からできること：
  • 部品をオーダーリストに追加するには、コールアウトをダブルクリクする。
  • 部品データウィンドウを表示するには、コールアウトをクリックする。
  • 画像の部品リストを表示するには、イラストの部品タブをクリックする。

 

インデックスの前のレベルに戻るには、 のアイコンをクリックします。
